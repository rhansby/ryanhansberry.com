BitBoard: A Web App for Collaborative Drawing

Background

BitBoard is a collaborative "digital whiteboard" web application that combines the advantages of the visual real estate of computer monitors and the fidelity of mobile device touch input. In short, BitBoard allows teams to collaboratively draw on a digital canvas on any device.

You (and friends) can try it out here, or check out the github page.

Below is a demo video of BitBoard in action:

BitBoard was developed as a quarter-long project for the CS130 (Software Engineering) course at UCLA, in Spring Quarter 2015. My partners and I are all proud of the resulting project, and I wanted to discuss the application featues which I think are the most unique, and that I think would be useful in other tools.

Unique Features

My favorite feature (which I implemented) is being able to see the screen bounds of a mobile user on a desktop view (notice that the iPhone's screen bounds are overlaid on the left desktop's view):

Another cool feature is BitBoard's boards' "infinite" nature; if a user draws close to a board's boundaries, the board will dynamically resize:

(This feature does display some small aberrations but for the most part works fairly well)
